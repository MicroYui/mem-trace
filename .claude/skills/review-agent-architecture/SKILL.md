## Goal

Review the architecture as an agent infrastructure system, not just as an ordinary backend project.

## Review Dimensions

Evaluate:
1. Agent loop and runtime lifecycle
2. Task/session model
3. Memory model
4. Context building and retrieval
5. Tool calling and tool registry
6. Skill system
7. Workflow orchestration
8. Persistence and state recovery
9. Concurrency and locking
10. Error handling and retry
11. Observability and traceability
12. Human-in-the-loop behavior
13. Security and permission boundaries
14. MVP feasibility
15. Resume/interview value
